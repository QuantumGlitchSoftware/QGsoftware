This Engine is a text based game engine with textures and chance mechanics. It is a very simple engine, but it gets the job done. This engine and the tutorial below were made so that even if you don't know how to code, you should still be able to use this engine. You use this engine by directly editing the code, you can open it with a text editor like Notepad or Brackets. Anything between square brackets [ ], including the square brackets themselves, is safe to edit and customize. Wherever "NOTE:" appears is a little more complicated, but there are tutorials for each "NOTE:" prompt below.
